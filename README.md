# SINPEN Thesis Project

**Clothing-company Inventory & Finance System**

**Stack:** Java 21, Gradle (Kotlin DSL), Spring Boot (optional), React (optional), Google Sheets integration, TiendaNube integration.

## Quickstart (developer)

Requirements:
- Java 21 installed
- Git
- Gradle wrapper (provided)

Build & run:
1. Clone the repo:
   `git clone git@github.com:youruser/yourrepo.git`
2. Build:
   `.\gradlew build`
3. Run (if Spring Boot app):
   `.\gradlew bootRun`
